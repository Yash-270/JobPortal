# 💼 Job Portal Platform

A full-stack job portal application that connects recruiters and candidates with role-based access control, secure authentication, and efficient job management features.

---

## 🚀 Features

- 👥 Role-based access (Admin, Recruiter, Candidate)
- 🔐 Secure Authentication (JWT + OTP Email Verification)
- 📄 Job Posting & Management
- 🧑‍💼 Candidate Profile Management
- 📎 Resume Upload & Download (Cloudinary)
- ⚡ Redis Caching (Optimized Performance)
- 🛡️ API Rate Limiting (Security)
- 🔍 Pagination & Filtering for Job Listings
- 📬 Automated Email Notifications

---
